📜 LeetCode em COBOL

  "se o sistema bancário global roda nisso, por que não resolver o leetcode?"

Este repositório contém minha resolução para desafio de LeetCode utilizando COBOL, compiladas com o GnuCOBOL. O objetivo é demonstrar domínio de lógica de algoritmos e estruturas de dados em uma linguagem orientada a processos e registros.


🛠️ Ferramentas e Requisitos

Para rodar estas soluções, você precisará do compilador GnuCOBOL (anteriormente OpenCOBOL).
Instalação

  Linux (Ubuntu/Debian): 
  
    sudo apt install gnucobol

  macOS:
  
    brew install gnu-cobol

  Windows:
    Utilize o MinGW-w64 ou o ambiente WSL.

🚀 Como Compilar e Executar

O COBOL não é uma linguagem interpretada como Python,Ruby ou PHP então você precisa compilar o código-fonte em um executável binário.

--> Compilar

    cobc -x twosum.cbl

--> Executar

    ./twosum
